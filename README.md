# VictoryTransportApp

A rewrite of transport-load application. A web app managing Loads of a Transportation company.

ASP.NET Web Forms, Bootstrap 4, JQuery Datatables, Ajax API calls, MS SQL 2016, Toastr, SweetAlert, Google Maps
